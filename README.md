🚀 Getting Started
Prerequisites
Modern web browser

Git (optional)

Installation
Clone the repository:

bash
git clone https://github.com/sahilgupta969/CyQuiz_App.git
cd CyQuiz_App
Open in browser:

bash
open index.html  # On macOS
 or
start index.html # On Windows
